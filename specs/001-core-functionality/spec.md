# 需求规格说明书：AI 文件管理器核心功能

**版本**: 1.0
**日期**: 2026-03-09
**状态**: Draft

---

## 1. 功能概述

AI 文件管理器是一个 macOS 桌面应用，用于集中管理和整理用户电脑上的 AI 相关配置文件（rules、agents、skills、commands 等）。

**核心价值**：
- 统一管理分散在不同目录的 AI 配置文件
- 快速搜索和定位文件
- 一键更新文件索引
- 支持自定义类别和路径

---

## 2. 用户故事

### US-001: 查看文件列表
**作为** AI 工具用户
**我想要** 按类别查看所有 AI 配置文件
**以便于** 快速了解我有哪些配置文件

**验收标准**：
- 侧边栏显示所有类别（rules、agents、skills、commands）
- 点击类别后，主区域显示该类别下的所有文件
- 文件列表显示：文件名、路径、修改时间、大小	

### US-002: 搜索文件
**作为** AI 工具用户
**我想要** 通过关键词搜索文件
**以便于** 快速找到需要的配置文件

**验收标准**：
- 顶部搜索框支持输入关键词
- 搜索结果实时更新（300ms debounce）
- 支持文件名模糊匹配
- 支持文件内容搜索
- 搜索结果高亮显示匹配内容

### US-003: 更新文件索引
**作为** AI 工具用户
**我想要** 手动刷新文件列表
**以便于** 获取最新的文件变化

**验收标准**：
- 顶部有"刷新"按钮
- 点击后重新扫描所有类别的文件
- 显示加载状态
- 更新完成后显示成功提示

### US-004: 管理类别
**作为** AI 工具用户
**我想要** 添加或删除文件类别
**以便于** 管理自定义的 AI 配置文件

**验收标准**：
- 通过对话框界面操作类别管理
- 可以添加新类别（名称 + 路径）
- 可以删除已有类别
- 可以编辑类别的搜索路径
- 配置保存到 config.json

### US-005: 自动更新文件索引（可选功能）
**作为** AI 工具用户
**我想要** 文件变化时自动更新列表
**以便于** 无需手动刷新即可看到最新文件

**验收标准**：
- 监控配置目录的文件变化
- 文件新增/修改/删除时自动更新列表
- 监控失败时降级到手动刷新模式
- 不影响应用的核心功能

## 3. 功能需求

### FR-001: 文件扫描
- 支持递归扫描指定目录
- 支持多个搜索路径
- 缓存扫描结果
- 支持增量更新（watchdog 监控）

### FR-002: 文件索引
- 索引字段：路径、名称、大小、修改时间
- 内存缓存索引数据
- 支持按类别分组

### FR-003: 搜索引擎
- 文件名模糊匹配（大小写不敏感）
- 文件内容全文搜索
- 限制搜索文件大小（< 10MB）
- 异步执行，不阻塞 UI

### FR-004: 配置管理
- 默认类别：rules、agents、skills、commands
- 默认路径：`~/.claude/rules`、`~/.claude/agents` 等
- 支持用户自定义类别和路径
- 配置持久化到 `~/.aifilemanager/config.json`

### FR-005: 用户界面
- 侧边栏：类别列表
- 主区域：文件列表（表格视图）
- 顶部：搜索框 + 刷新按钮

## 4. 非功能需求

### NFR-001: 性能
- 扫描 1000 个文件 < 2 秒
- 搜索响应时间 < 1 秒
- 内存占用 < 200MB
- CPU 空闲时 < 5%

### NFR-002: 可用性
- 界面简洁美观，符合 macOS 设计规范
- 支持键盘快捷键（Cmd+F 搜索、Cmd+R 刷新）
- 加载状态和错误提示清晰

### NFR-003: 兼容性
- 支持 macOS 11.0+
- 支持 Python 3.10+
- 支持 Markdown、JSON、YAML、Python、JavaScript 等常见文件格式

### NFR-004: 可维护性
- 代码模块化，职责单一
- 完整的类型注解
- 单元测试覆盖率 > 80%
